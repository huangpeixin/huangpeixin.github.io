# 为什么医生科普的那种医疗，在医院里很难体验到？

昨晚微博上收到一位妈妈的私信，内容如下：

“我家宝宝因为感冒咳嗽，分别在2月龄、3月龄、15月龄去了市妇幼，港大，市妇幼，三次就诊结果都是上呼吸道感染，医生分别开了美敏伪麻溶液，艾畅，右美沙芬。甚至在港大被开出艾畅时，是其刚注销了药品登记，我询问医生时他认为他没有错我不应该质疑他。因为知道这些药并不适合宝宝所以没有用，之后转诊了或者自行康复了。

市妇幼跟港大在深圳都算是比较好的医院，港大也以不乱开药而受到很多家长的喜爱。可是几次就诊的结果都让我很难相信医生跟医院，我想您的粉丝肯定也会有这样的问题。虽然包括您在内的很多医生跟机构都在做科普的事情这很好，但是就我的就诊经历来看，最后的结果总是因为接受了科普而质疑医生，我觉得这很奇怪，就是很多像我一样的妈妈最后都不信医院了，但看病不可能靠手机解决，医院也不应该因此失去公信力。

我相信在医院里的医生都是专业的，您也是在医院执业的医生，我想知道对此您有什么看法？或者如果我们面对科普，跟现实中的医生产生的矛盾，应该怎么解决？”

我相信有同样感受或者疑问的读者不少，其实这个问题在我开始写科普的时候就被人问到过，那时也写过一篇文章，以下是原文，分享给大家。

很多人说，为什么网上做科普的医生都这么好，告诉大家感冒药、咳嗽药不能乱给孩子吃，不能吃中药。而现实里那么多医生却乱开检查乱开药？网上觉得医生是王子，到医院发现都是青蛙，这是为什么呢？

让我们先假设一个现实，孩子感冒，到人山人海的急诊挂号排队大半天见到医生。然后家长问：我孩子发烧好几天，怎么办？医生答：发烧是好事，有助孩子对抗感染。家长再问：孩子还老咳嗽，看着好难受，有什么办法。医生：咳嗽是好事，有助于把痰液咳出避免肺炎。家长再问：孩子发烧咳嗽老不好怎么办？医生回答：让他多休息慢慢会好，有病情变化随时到医院来。然后医生会挨打吗？

我相信主动关注这个账号的家长大部分科学素养会比较高，而且看过我前面的那些科普，会更容易理解这种医生，所以大部分人的答案是不会，甚至还会觉得自己幸运，碰到一个好医生。

但要知道，在病人群体里，没有医学常识的还是大部分，有的根本就不关注健康知识，即便关注，也大多是关注中医养生之类，认为吃药打针才是真正治病的不在少数,这些人看着孩子生病，心急火燎的在吵杂喧闹的医院辛辛苦苦排了大半天的队，然后医生说这也是好事那也是好事，不打针也不开药就叫回家，不破口大骂算素质好的，大打出手肯定会有，这样的事情也发生过不止一起。

也有人说，医生如果耐心一点，把发烧、咳嗽为什么是好事，怎么处理详细解释一下，大部分家长还是可以理解的。是的，如果像我们在网上做科普一样，光一个发烧就写个好几篇文章，好几千字，医生blabla把这几千字花半个小时跟家长解释清楚，估计再暴躁的家长也会心平气和甚至喜笑颜开。

但问题是医生看一个病人5分钟轮到你都要大半天，那一个病人看半个小时你是不是等几天几夜了？到时候看病的不打医生，等着看病可能就出手了。那医院就不能多招一点医生吗？当然可以，挂号费十几二十块钱，有些地方还只有几块钱，看一个病人分到医生手里可能还不到一块钱，一个医生一天看一百多个病人收入才那么点，现在儿科医生辞职的都不少，再多招几个医生，一个医生一天看20个病人，那收入再降个几倍，还能招到医生吗？

即便是感冒，也并不是医生问个病史查个体就能百分百确诊，有些严重严重疾病的表现和感冒很相似，一旦漏诊误诊就可能出人命，在医患关系如此紧张的环境下，医生被打被杀经常有，不出错都可能被打，真要有什么出错了被打被骂也不会有人同情的。医院提供给医生的保卫措施也很有限，急诊室里一个医生被一圈病人围住是常态，有点紧急情况也是靠自己应对。查个血拍个片降低犯错的概率，顺着病人的要求下药是保护自己的方法，更何况查血拍片不需要自己出钱，还可以增加自己和医院的收入，何乐而不为？

中国的医学教育各式各样，决定了医生水平也是参差不齐，对诊断没那么有把握，胆小一点开一堆抗生素、抗病毒药物来个大包围式治疗，让自己安心一点；诊断很确定但认为感冒也应该用抗生素抗病毒药的也不在少数，多开药还可以增加一点收入，有的医院甚至有这样的引导政策，当然会有医生这样做。

再加上发展中医药是写入我国宪法的，别说有那么多中医院校培养了那么多中医生，普通的医学院校都开设有《中医学》这门课程，这些药物也都是政府认可的合法药物，非中医专业的医学教材里感冒的章节都推荐中医药治疗，很多医生自己感冒都吃一堆中成药，他们给病人开一堆也很自然。

所以，虽然我们在网上科普的时候告诉大家，如果不出现并发症，感冒是一个到时候自己会好的病，没有药物能治疗感冒，大多数发达国家都建议6岁以下的孩子不要吃这些感冒咳嗽药，因为既不能缓解症状，反而增加药物不良反应的风险。但你到了医院，验血拍片，然后开一堆中药西药，甚至打吊针都是很常见的，过几天感冒自愈了，医生家长皆大欢喜，这是最可能的现实。

所以在微博上有人说从来没有碰到假设中的那种医生，这是很有可能的，这样的医生有时被打，常常被骂，总是很穷，能坚持下去的自然很少。

网络是一个二维世界，医生科普不受其他因素的干扰，传播正确的知识才能受到更多的关注，所以大家觉得网上的医生好是自然的。而现实是多维的，行医要受到太多因素的干扰，在落后的医疗体制下，患者享受不到好的医疗服务是常态。贫瘠土地长出好果子是奇迹，长出歪瓜裂枣是正常的，梦想遇到现实就是如此。

梦想要成为真正的现实，需要民众有一定的科学素养能够理解医生看似不负责任实际却是正确的诊疗，社会需要一定的信任度才会相信医生正确的诊疗，需要法制环境保护医生的安全，也需要医学教育的改善来提高医生的水平，让医生的知识得到更新，需要改变以药养医的医疗体系，让医生可以靠诊金生活而不是靠卖药做检查来维生，但这些都是浩大的工程，不是单靠医改所能实现，也不是一朝一夕能完成的。（题图来自Pixabay）

看完这篇文章，可能有人会觉得你现在开私立诊所了，所以就写文章黑公立医院，但事实上这篇文章写于2014年，那时我还在公立医院上班，也没想过创业的事，但正是因为知道医疗行业的现状，而且短期内公立医院是很难改变的，后面才有了创业的想法，今天创建的怡禾，也就是想为改变这种现状做一点努力。

# 孩子脸上有白斑，需要吃打虫药吗？

俗话说，一白遮百丑，如果皮肤是均匀的白，确实可能给颜值加分。遗憾的是，有时不是均匀的白，而是皮肤出现一小块一小块的白斑，甚至出现在脸上，比如下图的这种，那就不是遮百丑而是让人忧心忡忡。

图片来自Disease Picuture

这是什么情况呢？是虫斑，要不要打虫呢？会发展为白癜风吗？

实际上，图中的白斑叫做白色糠疹，又叫单纯糠疹，这是一种主要发生于3-16岁的儿童的皮肤问题。这种病以斑片状色素减退为特征，通常分布于面部、颈部、躯干上部和四肢近端。

白色糠疹的白斑大小范围为直径0.5-5cm不等，边界清楚但稍不规则，且可有细小鳞屑。通常无症状，但可能伴瘙痒。日晒后，白色糠疹白斑周围的皮肤因为晒黑，因而与白斑形成明显反差而更加显得“白”。

会发展为白癜风吗？

很多家长也知道，白癜风也是一片片的发白，那这种白斑会不会是白癜风呢？或者担心这些白斑会不会发展为白癜风？

虽然两种疾病的外观都表现为白斑，但白癜风的白斑是乳白色，比白色糠疹看着更加“白”。典型的面部白癜风常发生在眼周和口周，颜色为均一的牛奶白或粉笔白，多为圆形、椭圆形，边界清楚。而白色糠疹通常发生在面颊部，最早期的皮损常为粉红色，数周后红斑消退，皮肤表面呈发白、干燥外观，边界不清楚，并且往往伴有细小的鳞屑。

白色糠疹属于色素减退性皮肤病，而白癜风是皮肤中功能性黑素细胞缺失，被认为是一种免疫性疾病，这与白色糠疹的发病原理完全不同，所以也不用担心会发展为白癜风。

要吃“打虫药”吗？

虽然民间普遍认为这种白斑为虫斑，认为和寄生虫，尤其是蛔虫有关，但目前白色糠疹是一种残余的炎症后色素减退性非特异性皮炎，它与特应性体质、日晒和沐浴的频率可能相关，与体内是否有蛔虫无关，因而不需要吃打虫药。

那要怎么治？

白色糠疹长在脸上，虽然看起来不那么美观，但一般不痛不痒，没有特别的症状。而且即便不治疗，也可以自行消退，如果不介意外观问题，我们完全可以等待其自行恢复，所以大可不必为此担心。

如果实在想治疗的话，可以局部用弱效糖皮质激素或钙调磷酸酶抑制剂和润肤剂可以减少干燥和鳞屑，可能促进复色，但这通常需要数月至数年。

我们也无法预知哪些孩子会出现这个问题，但现在认为可能和日晒、干燥有关，所以为了避免被这个问题困扰，一方面我们可以做好保湿，尤其是天气干燥的秋冬季，另一方面要注意防晒。

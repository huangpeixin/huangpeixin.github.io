# 怎样才能让孩子少上医院？

没有人愿意去医院，但现实是一旦有了孩子，家长们就经常要带孩子去医院。

孩子高烧不退，久咳不好，半夜肚子痛，觉得个子不够高，湿疹晚上痒得睡不着，不小心磕破了头出血，满口蛀牙影响吃东西，奶奶觉得孩子不爱吃饭看起来瘦 …..

因为这些问题，我们不得不经常去医院排队。验血、拍片、做雾化、打针…这是很多父母经历过的场景。

那有办法可以减少孩子去医院吗？

我在 之前的文章 里说过，怡禾诊所的口腔医生都感叹过，说来诊所看牙的孩子牙齿都太好了，一张口大多是整齐的小白牙，需要做补牙治疗的比较少，和公立医院里天天补牙形成了鲜明对比。

我们最近招聘的一个市场的同事，说我们的门诊市场工作应该不太好做，因为她自己作为怡禾的老用户，经过这么多的科普，对儿童常见病早就了然于心，给孩子吃药的次数都很少，更别说去医院了。

很多家长可能也会有这样的体会，养老大时什么都不懂，什么都担心，就三天两头跑医院找医生，等养老二、老三的时候，因为经历过，了解过，就从容多了，去医院的次数也少了。

其实大家也都能明白，孩子生病虽然次数更多，但大部分是小问题，而且很多是能自愈的感染性疾病，比如幼儿急疹、感冒、轮状病毒肠炎、手足口病等，或者是吃饭、睡觉、发育这些日常的小问题，遇到这些问题时更多是考验家长对这些问题的了解程度以及心理耐受力。

所以，要让孩子少上医院，我觉得主要就是做好两点：

1.建立正确的医学观念，知道怎么判断知识的对错，然后才能吸收到正确的知识。（ 如何建立可参看： 医生说法不一样，该听谁的？）

2.学习儿童常见健康问题知识

一方面能建立疾病和意外的预防意识，比如及时给孩子打疫苗，认真给孩子刷牙，及时去涂氟做窝沟封闭等，做好了儿童意外伤害防护等；

另一方面学习了相关知识，在遇到问题时自己能更淡定，也有底气去说服家人一起理性应对。

而要实现这两点，无非就是多学习，这里的很多读者依靠读公众号文章应该都已经做到了这两点，这是通过长年累月学习所实现的。

但公众号推送并不是体系化的知识，要建立一个养育孩子所需要的体系化知识，看公众号并不是一个很高效的方式。

所以怡禾做过一些系统化的育儿学习的产品，比如之前介绍过的包含0-36个月常见问题的 父母必修课2.0 。

# 十滴水可以治黑头吗？

答：不建议。

平时我们说的黑头，指的是黑头粉刺，是寻常痤疮的一种皮疹类型。

寻常痤疮是由于毛囊过度角化、皮脂分泌增多，毛囊内痤疮丙酸杆菌以及炎症共同作用导致的。

皮脂腺分泌的皮脂，和角化性物质在毛囊聚集，形成白头粉刺，毛囊因为持续被扩张而开放，角质形成细胞以及皮脂被氧化，再加上黑色素，让白头粉刺变成黑头，也就是开放性粉刺。

要治疗寻常痤疮，我们可以使用维A酸、壬二酸、水杨酸等抑制毛囊过度角化，也可以口服异维A酸控制皮脂分泌，有的还需要使用抗生素控制细菌增殖。另外，使用果酸、水杨酸这些浅表化学剥脱药物，对于黑头也有一定作用。

但因为这些药物不是自己买了用上就马上见效的，而是需要在医生评估病情做出针对性的方案，然后进行规范化治疗1-3月，才会起效。

十滴水成分包括樟脑、干姜、辣椒、桉油等，它的配料是酒精。这些成分里的酒精和桉油，或许能稍微溶解一点黑头上的皮脂，但这种效果是通过皮肤清洗、挤压也能实现的，这些效果都是短暂而不可持续的，而且这些成分还会破坏皮肤屏障，并可能刺激皮肤引起炎症反应，加重痤疮症状，所以不建议使用。

寻常痤疮影响外观，而且是一个慢性问题，容易给人造成压力。规范化的治疗需要比较长的时间，需要消耗人的耐心，所以有的人就会抓住大家的心理，提出一些简便易行又“快速有效”的偏方秘方，自然很容易受欢迎。

再加上痤疮本身可能受激素、精神压力等外部因素的影响，病情也会有自然波动，如果刚好自己缓解了，同时又用了这些未经验证的偏方秘方，就很容易被当成有效，然后形成口口相传。如果用了没效，会觉得反正是偏方，没用也正常，说出去还容易被人笑话，所以容易形成传播上的幸存者偏差，就造就了这些偏方的口碑。

和秘方偏方相比，前面说的那些规范化的治疗，是经过科学方法重复验证的，它们听起来更麻烦，更不神奇，但肯定比那些秘方偏方更可靠，更安全。

所以如果你被黑头、痤疮困扰，最可靠的办法是找皮肤专科医生就诊，而不是看视频刷到一个秘方就朝自己脸上抹。

# 一个儿科医生8年写成的「育儿宝典」，值得每位中国父母收藏

从2013年12月25日推送了第一条消息，这公众号到明天满8年。

哪怕是对人的整个一生，8年都是一段很长的时间，长到连我自己都很难相信自己坚持了这么久。

这8年时间里这个公众号累计推送了几千篇文章，几乎涵盖了儿童所有的健康问题，这些文章除了少数是其它专科医生撰写的外，大部分是我自己写的。

为了让这些知识更准确可靠，每篇文章的知识点反复核查，耗费了我非常多的心血，但也帮助了非常多的家长，让数以万计的孩子少吃了很多不必吃的药，实现了**「让父母少花钱，让孩子少受罪」**的宗旨，同时也改变了很多人的科学观念。

现在，很多家长遇到孩子健康问题，第一时间会到这里来查找答案，因为只有在这里查过才会觉得安心，这些文章也成了名符其实的“育儿宝典”，我也确信这些文章应该是我整个人生对社会所能贡献的最大的价值。

4年前起，我会把这个文章合集做一个年度更新，同时形成了下面这个目录，以方便大家收藏查看，今年依旧整理更新了，转发到朋友圈一定会有人受益。

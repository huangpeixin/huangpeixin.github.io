# 国家药监局也公告了，这些感冒药家长不要自己给幼儿吃

这个账号的老读者可能知道，我之前的文章说过很多次：

感冒药并不能让感冒好的更快，吃感冒药的目的是改善症状，而不是治好感冒。就儿童而言，感冒药改善症状的效果也不确切，相反还可能引发呼吸抑制等严重并发症。

所以很多国家很早就不建议给幼儿使用OTC的儿童感冒药，但我们国家却是有很多专门针对儿童的感冒药，这个现象我很多年前就讲了。

而且这些感冒药里很多都包含了退热止痛的成分，很多家长并不清楚这一点，把这些感冒药和退热药一起吃，导致对乙酰氨基酚过量造成肝损害等风险。这些问题我在之前的文章里也反复说过很多次。

也曾收到过很多读者的质疑，说不能用的药，国家为什么会批，医生为什么会开？不能一起吃国家为什么不提醒？当然也有人说我哗众取宠博人眼球等等。

再后来，艾畅这个药在2018年，强生公司自己注销了，参见： 别了，艾畅

昨天，我们国家药监局也终于发布了一份公告。

要求14个品种的感冒药的药品说明书进行修订。包括： 氨酚麻美口服溶液、氨酚麻美糖浆、小儿氨酚烷胺颗粒、氨酚伪麻那敏咀嚼片、小儿复方氨酚烷胺片、小儿氨咖黄敏颗粒、氨金黄敏颗粒、氨咖愈敏溶液、儿童复方氨酚肾素片、氨咖黄敏口服溶液、氨酚伪麻那敏分散片（Ⅲ）、小儿氨酚那敏片、小儿氨酚黄那敏片、小儿氨酚黄那敏颗粒 等14个品种。

修订的要求如下：

1.增加“不建议家长或监护人自行给2岁以下婴幼儿使用本品，应在医师或药师的指导下使用。”

2.增加“应严格按照药品说明书用法用量使用，避免用药过量。”

3.将“不能同时服用与本品成份相似的其他抗感冒药”改为“应避免合并使用含有相同或相似活性成份的抗感冒药。”

所以，现在我们国家药监局也终于作出了这样的要求了。这个公告来得有些晚，虽然只是限制了2岁，但也算一种进步。

之所以说这也是一种进步，并不是因为和我之前说的一样，而是这种警示是基于证据权衡了收益和风险的选择，是更符合患者利益的。

这里的老读者也应该知道，我先写完文章提醒，后续药监再限用或停用的药，也不仅仅是感冒药这件事，之前的匹多莫德、安乃近也都是如此。

# 孩子吃什么东西最有营养？

这东西大补，孩子吃了好。

这东西很贵，很有营养，给孩子多吃点。

这个东西富含多种氨基酸和微量元素，营养价值高，孩子吃了好。

.....

这类话我们可能听过很多，很多人可能也会好奇，这么多的食物，这么多的营养品，到底哪种东西最有营养？

这其实是一个伪问题，原因如下。

个体营养状况存在差异

对一个偏远山区吃不上饭严重营养不良的孩子来说，一碗米饭都非常有营养，因为能满足能量需求。而对一个发达地区的肥胖孩子来说，一碗牛肉都是营养负担的，因为它会让孩子变得更胖，增加高血压等问题的风险。

所以同一种食物，对不同的人的营养价值是不一样的。

不同阶段营养需求存在差异

对刚出生的孩子，除了维生素D，母乳就能满足孩子所有的营养需求，所以母乳对孩子就是最有营养的，但随着孩子生长，纯母乳提供的营养就慢慢跟不上孩子的需求了（不等于母乳没有营养了），比如到了4-6个月，如果不补充强化铁的米粉，肉类等食物，就可能出现缺铁性贫血。

所以同一种食物，对同一个人，不同阶段的营养价值也是不一样的。

营养不是越高越好

可能是因为经历过艰难年代，我们很多人还是担心营养不足，所以总是想给孩子吃高营养素的食物，但实际在营养上的追求的目标是均衡，而不是某一种营养素特别高。

比如猪牛羊肉富含蛋白质、脂肪，能满足人对很多营养素的需求，但如果光吃肉不吃水果蔬菜，同样会导致营养失衡。

食物的营养价值，少有不可替代性

虽然不同个体，不同阶段，营养需求会有不一样，但人大体的营养需求还是一致的，比如我们都需要有热量，需要蛋白质、碳水化合物、脂肪、维生素以及各种微量元素。

这些营养素，不同的食物富含的种类有所不同，但没有哪一种营养素是只有一种食物才能提供，而是都有可替代性。比如钙，我们从动物奶制品中获得，也可以从豆类及绿叶蔬菜中获得。

所以我们不需要太过于强调某一种食物的营养价值。

食物的作用是补充能量和营养素，不是治病

因为有“药食同源”的传统思想，我们给食物赋予了很多特殊的功效，所以很多人会寄望通过食物来调理身体或者治疗疾病，但这是不切实际的。

对于营养素缺乏所导致的疾病，比如营养不良、佝偻病、坏血病，我们可以通过能提供相关营养素的食物来纠正，但对绝大部分的疾病来说，还是要靠药物或其它方式来治疗，而不知指望某种特殊的食物。

所以，我们在食物上的追求，应该是合理搭配，均衡饮食。了解了这一点，我们就不需要把营养和疾病问题寄托在某一种特别的食物上了，也就不用想着给孩子吃什么最有营养这个问题了。

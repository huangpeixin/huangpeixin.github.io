# 知道孩子不可以用酒精擦身退烧，成人可以吗？

答：也不建议

就发热机制而言，儿童和大人其实没有区别，都是因为致热原传递的信号导致体温中枢调高了体温所致。

要退烧降温，根本在于致热原消除了，比如免疫系统控制了感染消除了致热原，体温中枢不再收到调高体温的信号，体温才会恢复正常。

包括擦澡、擦酒精、贴退热贴在内的措施，本质都是物理降温，是通过增加散热来降低体温，只要导致发烧的原因还在，只是增加散热的措施并不能达到持久降温的效果，只能短暂降低一点体温，而且会给人带来不适。

儿童因为皮肤薄，容易吸收酒精，再加上神经发育不完全，酒精可影响神经发育，大面积酒精擦浴婴幼儿还可能导致酒精中毒，所以不能使用。成人酒精擦浴因为皮肤吸收有限，而且大人基本能耐受这点酒精的影响，所以除非酒精过敏，否则不至于有什么危害，但对退烧没啥作用。

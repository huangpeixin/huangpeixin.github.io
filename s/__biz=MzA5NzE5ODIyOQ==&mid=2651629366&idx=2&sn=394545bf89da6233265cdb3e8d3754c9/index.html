# 课程 ｜孩子反复咳嗽，会不会是哮喘？

秋冬季节，孩子反复咳嗽很常见，有的可能碰巧连续感冒，有的可能是因为感冒并发了鼻窦炎导致症状持续，有的可能是因为哮喘之类的慢性问题。那么：

孩子反复咳嗽，怎么判断是不是哮喘？

孩子诊断哮喘了，要怎么用药管理？

哮喘长期用激素，会不会有影响？

哮喘急性发作了，要怎么办？

孩子有哮喘，如何避免急性发作？

怡禾诊所的孙杰医生在加入怡禾之前，是深圳港大医院的副顾问医生，也是儿童呼吸专科医生，目前他在怡禾的周末门诊，基本已经成为咳嗽专科，其中很多是哮喘的门诊管理病人。

所以我们请他筹备了一堂儿童哮喘管理课，从哮喘发生的原因、检查项目、激素用药、雾化吸入操作、日常护理、预防等多个角度来解答大家的疑惑。

为了让课程中的知识尽可能准确，制作课程过程中孙医生因为反复研读哮喘相关指南，还发现全球哮喘防治倡议（GINA）指南的一些问题，并写了邮件给编辑，可能还为明年指南的更新做了一点小小的贡献。

课程除了配合 PPT 的音频讲解，孙医生还以视频方式提供各种操作示范，包括家中各种雾化吸入装置的使用、肺功能做法等等，更直观呈现正确的哮喘治疗。

这次的课程内容由另外一位儿童呼吸科医生林汉炼审核，所以在知识的可靠度上应该很有保障，不足的地方可能是语音讲解部分会略显枯燥，但这些对内容对家有哮喘孩子的家长应该会很有用。
